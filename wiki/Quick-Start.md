# Quick Start

1. Run `npm install`.
2. Run `npm start`.
3. Open `http://localhost:3000`.
4. Open `http://localhost:3000/?demo=1` if you want sample transcript cards for visual testing.
5. Press `H` to hide the helper panel on the TV-facing screen.
6. Use manual lines first if the room is noisy or the browser speech engine is unavailable.
